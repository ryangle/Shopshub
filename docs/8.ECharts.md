2. Flex布局

    *  Flexible Box 的缩写，意为"弹性布局"
    * 任何一个容器都可以指定为 Flex 布局，Webkit 内核的浏览器，必须加上-webkit前缀
       
        ```css
        .box{  display: flex; }
        .box{  display: inline-flex; }
        .box{  display: -webkit-flex; display: flex;}
        ```
